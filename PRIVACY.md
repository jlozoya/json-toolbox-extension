# Privacy Policy for JSON Toolbox

Effective date: April 27, 2026

JSON Toolbox is a free and open-source Chrome extension for formatting, validating, inspecting, and transforming JSON directly in the browser.

## Data Collection

JSON Toolbox does not collect, sell, transmit, or share personal data.

The extension does not use analytics, tracking scripts, advertising networks, or external servers.

## Local Processing

All JSON formatting, validation, inspection, transformation, path extraction, and TypeScript type generation are processed locally in the user's browser.

JSON content is not sent to any external service.

## Local Storage

JSON Toolbox may store extension settings and local JSON history using Chrome's local extension storage.

This data remains on the user's device and is used only to provide extension functionality.

## Permissions

JSON Toolbox requests browser permissions only to provide its core functionality:

- `storage`: saves local settings and JSON history.
- `contextMenus`: adds a context menu option for opening selected JSON in JSON Toolbox.
- `tabs`: opens the full JSON editor from the popup, context menu, or JSON viewer.
- `scripting`: injects the JSON viewer into detected raw JSON pages.
- `webRequest`: detects JSON responses by reading response headers.
- Host permissions for `http://*/*` and `https://*/*`: allow the extension to detect raw JSON responses on websites and display the JSON viewer.

## Third Parties

JSON Toolbox does not share data with third parties.

## Contact

For questions about this privacy policy, contact the developer through the project repository.
